Background and Initial Motivation

The associahedron is an $n$-dimensional polyhedron whose facial structure represents the solution to several equivalent combinatorial enumeration problems. The $n$-dimensional associahedron has $C_n$ vertices, where $C_n$ is the $n$th Catalan number. There are many ways to understand the associahedron, and many are listed here, in its listing on the Hedra Zoo.

One important generalization of the associahedron is called the graph associahedron. If we consider a connected graph $G$, then connected subsets of vertices of $G$ are called tubes. There is a set of compatibility conditions for tubes, and then we say that a set of compatible tubes is a tubing. The set of tubings for any graph $G$ on $n+1$ vertices forms a simplicial complex, which is actually dual to an $n$-dimensional simple polytope, which we call the graph associahedron of $G$. The assocaiehdron we know is the graph associahedron of a path graph. Other interesting graph associahedra include the graph associahedron of a path graph, called a cyclohedron, and the graph associahedron of a complete graph, which happens to be the permutahedron.

Another important generalization of the associahedron is called the generalized associahedron. These were introduced by Fomin and Zelevinsky to describe finite type cluster algebras. Every irreducible crystallographic root system of type $\Phi$ defines a cluster algebra of type $\Phi$, and every cluster algebra of type $\Phi$ then defines an associahedron of type $\Phi$. There are many crystallographic root systems; the root system of type $A_n$ is associated with the symmetry group of a simplex, and the root systems of type $B_n$ and $C_n$ are associated with the symmetry group of a hypercube. The type $B_n$ associahedron is combinatorially isomorphic to the cyclohedron.

My thesis research was initially motivated by the search for a "type $B$" generalization of graph associahedra. The main takeaway is that graph associahedra can be realized by repeatedly truncating faces of a simplex, and the symmetry group of an $n$-dimensional simplex is $A_n$. Because the symmetry group of an $n$-dimensional hypercube is $B_n$, it makes sense to see what happens when faces of a hypercue are repeatedly truncated. As a result, this provides a very natural "type $B_n$" generalization of the graph associahedron, which we call the hypercube graph associaehdron.However, I found a strange result: for lower dimensions, I conjecture that the cyclohedron cannot be realized as a hypercube graph associahedron. Instead, I found a large number of different interesting polyhedra defined by different graphs on vertices $-n,\ldots,-1,1,\ldots,n$.

Research Methodologies

The work in my thesis can be roughly divided into four categories:

  1. Define a theoretical framework for understanding hypercube graph associahedra. This relied heavily on the definitions of graph associaehdra, nested complexes, and building sets defined by other authors. This work is primarily a translation of existing work into a specific level of generality.
  2. Find graphs with interesting hypercube-graph associahedra. Many were found by checking graphs with certain symmetries and basic properties (paths, cycles, complete graphs, etc.) but some were found by computer search. Even though I did not find any higher-dimensional hypercube graphs whose hypercube-graph associahedra were isomorphic to the cyclohedron, I found several which shared an f-vector with the hypercube graph associahedron.
  3. Write code in SageMath which can generate hypercube-graph associahedra, as well as calculate their f-vectors via brute-force. The images on this site were generated with this code.
  4. Develop enumeration techniques which recursively calculate the f-vectors of facets of hypercube-graph associahedra. These formulas, when applied to power series, tend to yield partial differential equations which are hard to solve but easy to verify. These methods also translate very easily to other graph associahedra.
